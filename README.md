# AutoCar Blog Client

This is the **frontend client** of the AutoCar Blog application built with **Next.js**, **React**, and **Tailwind CSS**. It provides a user-friendly interface to browse, sell, and manage cars.

---

## Project Overview

- **Framework:** Next.js 16
- **Language:** JavaScript (React 19)
- **Styling:** Tailwind CSS 4 + DaisyUI
- **State & Utilities:** Axios, js-cookie
- **Firebase:** Authentication & database integration
- **UI Components:** Swiper for sliders, React Icons
- **Alerts:** SweetAlert2

This project allows users to:

1. Browse all available cars.
2. View car details.
3. Sell a car by filling a form.
4. Login/Register and manage session.
